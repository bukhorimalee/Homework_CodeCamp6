1.Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น table ก็จะแสดงผลในส่วนของตาราง
    Selector  {property: value}  table {color: red;}    
2.Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษ
    table { color : red }
3.Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt
    p { font - family : "sans serif" }

  • selector มีกี่แบบอะไรบ้าง
    • 1. Class Selectors Class Selectors เป็น Selector ที่เราต้องกำหนดชื่อของ Selector ขึ้นมาเอง โดย Class Selector การเรียกใช้งานต้องระบุชื่อ Class Selectors ให้กับ Tag ของ HTML จึงจะสามารถใช้งานคุณสมบัติของ Class ที่ได้สร้างไว้ได้ 
    <style>
        .myclass{
            font: bold 12px Tahoma;color: #009;
        }
        </style>
    2. ID Selectors ซึ่งเขียนอยู่ภายใน Tag ของ HTML โดยที่ Tag ของ HTML ใดมี ID ซึ่งตรงกับ ID Selectors ที่กำหนด คุณสมบัติของ CSS ที่กำหนดไว้กับ Selector ก็จะมีผลต่อ Tag นั้น 
    <style>
            #myid{
                font: bold 12px Tahoma;color: #009;
            }
            </style>
    3. TAG Selectors สำหรับ Selector ในรูปแบบสุดท้ายที่จะกล่าวถึงในเบื้องต้นนี้เป็น Selector ที่อ้างอิงกับชื่อ Tag ของ HTML โดยตรง เช่น 
    <style>
                h1{
                    font-family: Tahoma;
                }
                </style>

value ใน display property มีอะไรบ้าง grid และ flex แตกต่างกันอย่างไร justify and align แตกต่างกันอย่างไร 
{display: none;} = ไม่ตั่งค่า
{display: inline;} = เป็นค่าเริ่มต้น 
{display: block;} = แถวเดียวกัน
{display: inline-block;} = ซ่อนกัน
Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน Grid เราสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout เช่นตัวอย่างตามรูปด้านล่าง เราบอก grid ว่าให้ layout ของ header เต็ม row บรรทัดต่อมาให้ menu เป็น 2 columns content เป็น 8 columns อะไรประมาณนั้น
